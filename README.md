# Secure Coding Review - Python Packet Sniffer

## Overview
This project demonstrates a secure coding review of a Python packet sniffer application using Scapy.

Two versions are included:
1. Vulnerable Version
2. Secure Version

## Features
- Packet Sniffing
- Protocol Filtering
- Blacklist Detection
- Suspicious Port Alerts
- Secure Logging
- Exception Handling
- CLI Support

## Technologies
- Python
- Scapy

## Installation

pip install -r requirements.txt

## Usage

python secure_sniffer.py

python secure_sniffer.py -p TCP -c 10

- vulnerable_sniffer.py
- secure_sniffer.py
